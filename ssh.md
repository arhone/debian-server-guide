# Подключение к серверу по ssh
Добавляем свой ключ, что бы заходить по ключу, а не по паролю
```
ssh-copy-id root@a-0.example.com
```
Вводим пароль

Дальше входим уже без пароля
```
ssh root@a-0.example.com
```
